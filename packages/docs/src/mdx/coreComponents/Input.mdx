## TimeInput

Time input component

The time input uses a text input box to
write time, the input forces the user to enter
in a specific format e.g. (hh:mm:ss).

When variant `hour12` is `false` the time input will act as a 24 hour input.

When property `hour12` is `true`, the input is
postfixed with am/pm, and the hours can only go
from 01 to 12, there is also a select besides the input component
that lets the user to choose between AM and PM.

Duration input component (DurationInput)

Is a variant of TimeInput but the highest time unit
in the input are not limited in value (can go up to max 99).
Also big difference between Duration and TimeInput is that value in
DurationInput is handled with seconds only.

### Demo 123

export const TimeInputDemoComponent = ({}) => {
  const [value, setState] = useState(0)
  const onChange = useCallback(
    newState => {
      setState(newState)
    },
    [value]
  )
  const [timeValue, setTimeValue] = useState({ hour: 0, minute: 0, second: 0 })
  const onTimeChange = useCallback(
    newState => {
      setTimeValue(newState)
    },
    [value]
  )
  return (
    <>
      <Typography variant="default-text">
        hour24 variant, twenty-four format hours, minutes and seconds
      </Typography>
      <SpaceBlock variant={8} />
      <TimeInput
        value={timeValue}
        formatLabel="HH:MM:SS"
        onChange={onTimeChange}
      />
      <SpaceBlock variant={24} />
      <Typography variant="default-text">
        hour12 variant, AM/PM hours and minutes
      </Typography>
      <SpaceBlock variant={8} />
      <TimeInput
        value={timeValue}
        hour12={true}
        formatLabel="HH:MM:SS"
        onChange={onTimeChange}
        hour12MeridiemLabels={{ am: 'AM', pm: 'PM' }}
      />
      <SpaceBlock variant={8} />
      <DurationInputField
        value={value}
        format="hh:mm:ss"
        unitLabel="hh:mm:ss"
        label="Duration variant with seconds"
        onChange={onChange}
      />
    </>
  )
}

### Basic usage

#### Normal 24 hours, minutes and seconds

```typescript type=live
<TimeInput
  value={{ hour: 0, minute: 0, second: 0 }}
  onChange={newState => {}}
/>
```

#### Normal AM/PM hours and minutes

```typescript type=live
<TimeInput
  value={{ hour: 0, minute: 0 }}
  hour12={true}
  onChange={newState => {}}
/>
```

#### Duration variant with minutes and seconds

```typescript type=live
<DurationInput value={0} format="mm:ss" onChange={newState => {}} />
```

#### Duration variant with hours, minutes and seconds

```typescript type=live
<DurationInput value={0} format="hh:mm:ss" onChange={newState => {}} />
```

#### Duration variant with seconds

```typescript type=live
<DurationInput value={0} format="ss" onChange={newState => {}} />
```

#### WithField variant with seconds

```typescript type=live
<TimeInputField
  label="There is also a TimeInputField version"
  value={{ hour: 0, minute: 0, second: 0 }}
  hour12={true}
  onChange={newState => {}}
/>
```
